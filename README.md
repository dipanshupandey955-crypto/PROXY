# 🎮 Tic Tac Toe Game in C++

A simple console-based **Tic Tac Toe** game developed in **C++**. This project allows two players to play alternately in the terminal. It is a beginner-friendly project that demonstrates the use of arrays, functions, loops, conditional statements, and basic game logic.

---

## 📌 Features

* Two-player gameplay (Player X and Player O)
* Input validation
* Win detection
* Draw detection
* Simple and clean console interface
* Beginner-friendly C++ code

## 🎮 How to Play

1. The game board contains positions numbered **1 to 9**.
2. Player **X** starts the game.
3. Players take turns entering a position number.
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all nine positions are filled without a winner, the game ends in a draw.
